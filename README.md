#  GymTracker https://gym-project-cipz.onrender.com/
 
A full-stack Flask web application for tracking workouts, monitoring fitness progress, and getting AI-powered coaching — all in one place.
 
## Features
 
- **User Authentication** — Secure signup, login, and logout with bcrypt password hashing
- **Profile Setup & Editing** — Personalized onboarding with fitness level, goals, health metrics, BMI/BMR/TDEE calculations
- **Daily Workout Logging** — Pre-workout check-ins and post-workout result tracking
- **Progress Analytics** — Streaks, consistency scores, heatmap calendar, charts, and session history
- **Nutrition Meal Plan Prediction** — ML-powered meal and exercise plan recommendations
- **AI Chatbot** — Page-aware assistant that answers questions based on the current page context
- **Dark / Light Theme** — Persisted theme toggle across sessions
---
 
## Tech Stack

 Backend : Python, Flask, SQLAlchemy, Flask-Migrate (Alembic) 
 Database : MySQL (via PyMySQL) 
 Auth : Flask-Login, bcrypt 
 AI / LLM : Groq API 
 ML Models : scikit-learn (joblib) 
 Frontend : HTML ,JS, CSS
