Features

Authentication system

Authentication system with secure password hashing using bcrypt library

Workout Tracker

tracks the streak and workout log sets

progress dashboard & Consistency Score

record all the data of user and give the Consistency score based on no of days login and streak etc

pre-Workout Review and post-workout review

user can give post and pre workout data to the system

Ai feedback

llm gives feedback to the user based on the user data collected form the pre and post workout form

AI chatbot

page aware llm chatbot that can unteract and give answer to the users questions related to that page

BMI Calculator

in profile page system automatically calculate and display the bmi from the user data

Meal Plan Predictor

the machine learning model predicts the meal plan using the user data like Gender , goal,bmi ,exercise schedule

Blog Articles

articles with photos and videos related to the exercise set so that user can understand how to do and why do the exercise set properly

Daily Quote

daily quote is displayed in homw page to motivate the users

user profile

user can update and delete the data related to the user such as goal , weight , height etc

account deletion and creations

no two user can have same username and if user wants to delete the account user can delete there account from the profile page
